# Clear Morning (Personal Fork)

This is a personal fork of [Clear Morning](https://github.com/saltyaom/clear-morning) by @saltyaom.  
It is modified for individual use with the following changes:

-   Added a **background blur effect** when using the search bar
-   Included some **personal pinned websites** for quick access
-   Added `manifest.json` to use the local HTML file as the new tab page in **Edge Developer mode**

> Note: This fork is intended for **individual use only**.
